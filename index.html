How to use this file

  1. Pick the AHJ district section.
  2. Open each checklist group and check off what is included in your package.
  3. Separate permit/plan routes from annual/ITM/report routes before sending anything.
  4. Follow the linked source documents/portals.
  5. Use the naming template suggestions before upload.
  6. Flag every yellow “Verify” item with the AHJ/plan reviewer.

Checkbox progress is saved locally in your browser (localStorage).

Package architecture (recommended master stack)

Core submission stack:
  1. Application / permit intake record
  2. Plans / shop drawings (ERCES/BDA/DAS)
  3. Engineering calculations + link budget
  4. Cutsheets / listings (BDA, DAS, antennas, coax, UPS, etc.)
  5. RF grid testing docs (pre/post as required by AHJ)
  6. Narrative / compliance matrix (IFC/NFPA + local amendments)
  7. Battery/UPS and monitoring details
  8. Response letter for resubmittals (when applicable)

Expansion queue (next districts / recovery audit queue)

  • Poudre Fire Authority (Fort Collins / Loveland area)
  • Aurora Fire Rescue (city + county coordination checks)
  • Colorado Springs Fire Department / Pikes Peak-area AHJs
  • Arapahoe County / municipal overlays (Cherry Hills, Centennial, etc.)

Keep each new district section in the same structure: permit route + report route + codes + checklist + naming + verification gaps.
